# ¿Que es un CRUD?

Titular: Niko

**CRUD** son las **acciones básicas** para trabajar con datos:

- **C**: Create (Crear)
- **R**: Read (Leer)
- **U**: Update (Actualizar)
- **D**: Delete (Eliminar)

En Laravel, un CRUD se arma combinando:

- **Rutas**
- **Controlador**
- **Modelo**
- **Vistas**
